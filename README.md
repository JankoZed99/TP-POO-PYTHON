## Objectif

Construire progressivement un système représentant des boissons dans un café en utilisant :

- Programmation Orientée Objet
- Classes abstraites
- Pattern Décorateur
- Surcharge d'opérateur (+)
- Data Classes
- Héritage et polymorphisme
- Héritage multiple

## Fonctionnalités

- Création de boissons (Café, Thé)
- Ajout d’ingrédients (Lait, Sucre, Caramel)
- Combinaison de boissons
- Gestion des commandes
- Gestion des clients
- Programme de fidélité
